📚 EVV Report Generator - Complete Guide

🎯 Purpose

This enterprise-grade application merges billing data with employee slot (EVV) data and client information to generate comprehensive reports. It identifies discrepancies, handles multi-provider scenarios, and ensures accurate billing with automatic data normalization and validation.

📋 Required Files

You need three Excel (.xlsx) files:

  1. Billing Data File - From Therap Billing Data
  2. Slot Report File - From Therap Scheduling/EVV Reports
  3. Client By Status File - From Focus or shared drive with MAIDs

🚀 How to Use

  1. Load Files: Use the browse buttons or drag all three files into the drop zone.
  2. Select Output Mode: Full (all columns) or Basic (customizable subset).
  3. Merge Data: Click the Merge button to process and validate.
  4. Review Issues: Check the "Rows with Issues" card and highlighted rows.
  5. Export: Save the professionally formatted Excel report.

🔒 Data Privacy

This application runs entirely in your browser. No data is sent to any server. All processing happens locally on your computer, ensuring complete privacy and HIPAA compliance.
